# User Management - Spring Boot + Thymeleaf

##  Description

Ce projet est une application web CRUD simple développée avec :

- 🌱 Spring Boot
- 🗄️ Spring Data JPA
- 🎨 Thymeleaf
- 🛢️ MySQL
- 🎨 Tailwind CSS (UI moderne)

L’application permet de :
- ➕ Ajouter un utilisateur
- 📋 Afficher la liste des utilisateurs
- ✏️ Modifier un utilisateur
- ❌ Supprimer un utilisateur

---

## ⚙️ Technologies utilisées

- Java 17
- Spring Boot 3.x
- Spring Data JPA
- Thymeleaf
- MySQL
- Maven
